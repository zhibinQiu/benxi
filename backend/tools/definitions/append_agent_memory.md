append_agent_memory — 在用户的长期记忆中追加一条笔记。

## When to use

- 用户明确要求你"记住"某条信息（如"帮我记住……"、"记下来……"）
- 用户希望保存一段文本到自己的长期记忆中供以后回顾

## When NOT to use — 极其重要

- **不得用本工具作为你无法执行用户请求时的兜底手段**。如果你没有能力绘制流程图、生成图片、执行计算等，必须直接告诉用户"抱歉，我暂时无法绘制流程图"，而不是把请求内容写入记忆。
- **不得代替其他工具**。用户请求调用某个技能、搜索信息、执行操作时，调用对应的工具，不要写入记忆。
- **不得用本工具来"假装已完成任务"**。写入记忆不等于完成了用户的原始请求。无法完成时如实说明。
- **不要记录你已经执行的工具调用结果**。工具结果会自动保存在对话历史中，不需要手动写入记忆。

## Returns

- 工具执行结果（具体返回字段由 tool schema 定义）
