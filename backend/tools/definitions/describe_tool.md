查看任意工具的完整定义：描述、参数 JSON Schema、使用示例。
调用后该工具会在下一轮对话变为可用，直接调用即可——不需要手动拼接 JSON。

## When to use
- 你知道需要用什么工具，但不清楚参数结构
- 你想查看某个工具的完整定义（所有参数、类型、必填项）
- 你想通过工具名搜索确认哪个工具适合当前任务
- 工具不在当前可见列表中，需要先加载其定义

## When NOT to use
- 你已清楚工具参数结构（直接调用，无需 describe）
- 你只需要搜索 Skill（用 find_skills）
- 你不知道工具名（先看 tools.md 路由索引）

## Returns
- 工具完整定义：描述、参数 JSON Schema、使用示例
- 该工具已被注册为「已发现」，下一轮对话中直接可见可用
