# 常见问题

## 如何创建管理员账号？

系统启动后，首次注册的账号为 bootstrap 管理员。

## 支持哪些 PDF 格式？

支持所有标准 PDF 文件，包括扫描件（需 OCR 插件）、科学论文、技术文档等。

## 知识库支持哪些文档格式？

- PDF
- Markdown
- TXT
- 通过 API 接入的文本

## AgentKit 是什么？

AgentKit 是本析平台的多智能体架构 Python 工具包。详见 [AgentKit 仓库](https://github.com/zhibinQiu/Agentkit)。

## 开源协议是什么？

本项目采用 [AGPL v3](LICENSE) 协议。
