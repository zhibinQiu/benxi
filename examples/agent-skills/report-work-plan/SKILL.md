# 工作计划

## 适用场景

- 年度/季度/项目工作计划、工作方案、行动方案、里程碑计划
- 用户提到「工作计划」「工作方案」「行动计划」

## 工作流程

1. 明确计划周期、责任主体、总体目标。
2. 按需 `knowledge_retrieve` / `web_search` 获取政策依据或背景。
3. 按 `templates/outline.md` 输出，任务可量化、可验收；时间线清晰。

## 章节结构

见 `templates/outline.md`。
